# Calculator_App
Javascriptで電卓アプリを作る

## 機能
-  整数及び小数の四則演算(+-*/)ができる
-  符号の四則演算(+-*/)ができる
-  00の入力で100・200,0.001といった入力ができる
-  ACでリセットできる

## 仕様
- ×と÷は優先して計算
- ACや何も入力されていない時は空列ではなく「0」を表示。
- 0で割ると、「0では除算できません」が表示される
- 「0では除算できません」の後に1~9を入力すると数字が上書き、それ以外のボタンを押すと0を表示

# エラー制御
- 何も入力されてない状態で0を押すと0、その後00を押すと何も変化しない
- 2× 3÷ で00 小数点の入力を制御
- +-*/と小数点の入力が連続でされないようにする
- 0の時に-(マイナス)符号として入力できるようにする +×÷はしない
- -(演算子)の後に-(符号)を入力できるようにする
- *はx,/は÷で入力できるようにする
